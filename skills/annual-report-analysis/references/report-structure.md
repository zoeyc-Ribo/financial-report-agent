## FORMATTING REQUIREMENTS

### 1. Page 1 Requirements
- Clear overall conclusion
- Valuation view or price target shown clearly if applicable
- Summary table with key annual metrics
- 3-4 paragraph-length bullets with ■ character

### 2. All Tables Requirements
- Source line at bottom
- Clear column headers
- Shading for header rows
- Focus on key comparisons, not exhaustive reproduction of statements

### 3. All Charts Requirements
- "Figure X - [Title]" caption above
- "Source: [Source]" line below
- Professional styling
- Chart must support a point made in the text

### 4. Year Notation
- Use A for actual where useful (FY2024A)
- Use E for estimate where useful (FY2025E)
- Clearly distinguish reported actuals from guidance / estimates / scenarios

### 5. Writing Style
- Lead with numbers ("Revenue grew 15% to $1.2B" not "Strong revenue growth")
- Use "vs." not "versus"
- Be direct and concise
- Focus on what changed this year
- Separate reported facts from forward-looking commentary

### 6. Hyperlink Requirements
- All URLs must be clickable hyperlinks in Word
- Use blue, underlined text that opens on Ctrl+Click
- Display text should be meaningful, not a raw URL
- Every major source citation should have a clickable link where applicable
- Do not leave plain text URLs in the document body unless explicitly requested
