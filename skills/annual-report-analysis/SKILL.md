# Annual Report Analysis

Create professional **ANNUAL REPORT ANALYSIS REPORTS** based on a company's latest annual results and annual filing, following institutional financial research standards.

**Key Characteristics:**
- **Length**: 8-12 pages
- **Word Count**: 3,000-5,000 words
- **Tables**: 1-3 summary tables (NOT comprehensive)
- **Figures**: 8-12 charts
- **Audience**: Readers familiar with the company who need a current annual update
- **Focus**: What changed this year - annual performance, segment trends, financial quality, outlook, and thesis impact
- **Font**: Times New Roman throughout (unless user specifies otherwise)

## When to Use

Use when the user requests:
- "Analyze [Company]'s annual report"
- "Create a FY2024 annual report analysis for [Company]"
- "Review [Company]'s 10-K"
- "Analyze [Company]'s full-year results"
- "Write an annual financial analysis report for [Company]"

**Do NOT use if:**
- User requests "initiation report" -> Use a different skill
- User requests "flash note" or "quick take" -> Use a different format
- User requests private data analysis without accessible source materials

## Critical Requirements

### 1. Annual Data and Timeliness
- Use the latest available annual-specific materials.
- Focus on the requested or latest fiscal year.
- Do not rely on stale memory or outdated filings.
- Focus on what is NEW in the current year rather than repeating generic company background.
- Clearly distinguish final filed annual data from management outlook or market expectations.

### 2. Performance Analysis
- Lead with the most important annual performance conclusions.
- Quantify year-over-year changes in revenue, EPS / net income, margins, and cash flow.
- Compare results against expectations or guidance where reliable sources are available.
- Explain WHY results differed from expectations, management targets, or prior-year trends.

### 3. Summary Format
- Keep tables to 1-3 (summary only, not comprehensive).
- Do not reproduce the full P&L / Cash Flow / Balance Sheet unless the user explicitly asks.
- Focus on decision-useful metrics and major changes.
- Assume the reader does not need a full initiation-style company primer.

### 4. Citations and Source Attribution

**CRITICAL**: Properly cite all data with SPECIFIC sources and CLICKABLE HYPERLINKS.

**Example source format for figures and tables:**

Source: FY2024 Form 10-K filed February 21, 2025; Company annual earnings release  
Hyperlink "Form 10-K" to the SEC filing  
Hyperlink "annual earnings release" to the company IR results page

**How hyperlinks should appear in Word:**
- Document names appear as blue, underlined clickable links
- Reader can Ctrl+Click to open source directly
- Do not use plain text URLs when a formatted hyperlink can be used

**Required sources list:**
Cite in every annual analysis:
- Annual earnings release / annual results announcement (with date and URL)
- 10-K / 20-F / annual report (with filing date and link)
- Earnings call transcript / webcast / prepared remarks (with date, if available)
- Investor presentation / supplemental materials (if available)
- Consensus estimates / expectation source (with date, if available)
- Prior-year filing or prior guidance source when discussing changes over time

**Required Sources section at the end of the report:**

SOURCES & REFERENCES

Annual Results Materials (FY2024):
- Annual Earnings Release (February 20, 2025)
- Form 10-K (Filed February 21, 2025)
- Earnings Call Transcript (February 20, 2025)
- Investor Presentation (February 20, 2025)

Each line above should be formatted as a clickable hyperlink in the final DOCX report.

**Verification checklist:**
- [ ] Every figure has a source with specific document and date
- [ ] Every table has a source with a document reference
- [ ] Expectations / guidance analysis cites the source with date
- [ ] Outlook changes cite both current and prior source materials when relevant
- [ ] Key statistics have footnotes where needed
- [ ] Sources section lists all materials with URLs or hyperlinks
- [ ] All URLs are clickable hyperlinks, not plain text
- [ ] All SEC filings are hyperlinked to EDGAR or official filing pages

### 5. Estimate and Expectation Handling
- Update the analysis based on annual results, management outlook, and estimate implications.
- Show old vs. new assumptions clearly when relevant.
- Explain what changed and why.
- If premium estimate sources are unavailable, do not fabricate consensus.
- Fall back to company guidance, prior-year comparisons, multi-year trends, and management targets where available.

## High-Level Workflow

The annual report analysis process follows 5 phases:

### Phase 1: Data Collection and Verification

**CRITICAL: TRAINING DATA MAY BE OUTDATED**

**Before starting, complete these 4 steps in order:**
1. Check today's date.
2. Search for latest annual materials using:
   - "[Company] latest annual report"
   - "[Company] latest annual results"
3. Verify the fiscal year.
4. Check supporting materials such as transcript / webcast / investor presentation dates.

**Common mistake:** Using outdated annual filings or prior-year materials instead of the latest available annual documents.

**Requirements:**
- Search for the latest annual materials - do NOT rely on training data
- Record today's date and the filing / release dates found
- Verify the fiscal year is correct
- Verify transcript / webcast / presentation dates align with annual results
- If materials are incomplete or outdated, search again and state the limitation clearly

See `references/workflow.md` for detailed search procedures and verification steps.

### Phase 2: Analysis
- Annual revenue, earnings, and margin analysis
- Segment / geography / product line breakdown where relevant
- Cash flow and balance sheet quality analysis
- Management outlook and strategy analysis
- Estimate implications and thesis update

See `references/workflow.md` for detailed analysis framework.

### Phase 3: Chart Generation
Create 8-12 charts focusing on annual trends and what changed this year:
- Annual revenue progression
- Annual EPS or net income progression
- Margin trends
- Revenue by segment / geography
- Operating cash flow and free cash flow trends
- Debt, liquidity, or leverage trends where relevant
- Expectations / guidance comparison
- Valuation charts if relevant

See `references/workflow.md` for chart specifications.

### Phase 4: Report Creation
Create an 8-12 page DOCX report with a specific structure.

See `references/report-structure.md` for complete page-by-page templates and formatting requirements.

**High-level structure:**
- Page 1: Executive summary and overall thesis view
- Pages 2-3: Annual results analysis
- Pages 4-5: Business performance and financial quality
- Pages 6-7: Management commentary, outlook, and thesis update
- Pages 8-10: Valuation, risks, estimate implications, and conclusion
- Pages 11-12: Appendix (optional)

### Phase 5: Quality Check and Delivery
Verify content, formatting, accuracy, and source quality before delivery.

See `references/best-practices.md` for quality checklist and common mistakes to avoid.

`Nike_FY2024_Annual_Report_Analysis.docx`

**Contents:**
- Page 1: Executive summary and key takeaways
- Pages 2-3: Annual results analysis

- Pages 6-7: Management commentary, outlook, and thesis update
